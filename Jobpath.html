Project Details

Overview

Job Path is a career-guidance mobile application that matches users' certified skills with relevant job roles in real time. It analyzes uploaded certificates and provides personalized career insights, job suggestions, and skill gap recommendations.

Objectives

  • Matches user skills with suitable job roles automatically
  • Extracts skills from uploaded certificates
  • Suggests top job roles based on real-time market demand
  • Includes a chatbot for career guidance and skill gap insights
  • Built using React Native, Node.js, Express, and MongoDB
